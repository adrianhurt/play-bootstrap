Play 2.6.2 and Bootstrap 4 (alpha) support

The project has been split to give support for Play 2.6.2 and Bootstrap 4 alpha. So now there are 6 parallel versions depending on which version you need for Play Framework and Bootstrap:

  • 1.2-P26-B3: Built for Play 2.6.2 and Bootstrap 3.
  • 1.2-P26-B4: Built for Play 2.6.2 and Bootstrap 4 alpha.
  • 1.2-P25-B3: Built for Play 2.5.16 and Bootstrap 3.
  • 1.2-P25-B4: Built for Play 2.5.16 and Bootstrap 4 alpha.
  • 1.2-P24-B3: Built for Play 2.4.11 and Bootstrap 3.
  • 1.2-P24-B4: Built for Play 2.4.11 and Bootstrap 4 alpha.

Note! There is a selector on the topbar's right side for change between all versions.
