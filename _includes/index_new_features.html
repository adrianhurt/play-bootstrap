Play 2.8.1 support (with Scala 2.13)

The project has been split to give support for Play 2.8, 2.7 and 2.6 and Bootstrap 4 and 3. So there are 6 parallel versions depending on which version you need for Play Framework and Bootstrap:

  • 1.6.1-P28-B4: Built for Play 2.8.1 and Bootstrap 4.
  • 1.6.1-P28-B3: Built for Play 2.8.1 and Bootstrap 3.
  • 1.6.1-P27-B4: Built for Play 2.7.3 and Bootstrap 4.
  • 1.6.1-P27-B3: Built for Play 2.7.3 and Bootstrap 3.
  • 1.6.1-P26-B4: Built for Play 2.6.23 and Bootstrap 4.
  • 1.6.1-P26-B3: Built for Play 2.6.23 and Bootstrap 3.

Note! There is a selector on the topbar's right side for change between all versions.
